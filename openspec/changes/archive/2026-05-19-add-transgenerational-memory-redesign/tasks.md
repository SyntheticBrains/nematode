## 1. Scaffold sensory-conditional substrate (Commit 1)

- [x] 1.1 Extend `quantumnematode/agent/transgenerational_memory.py`: add optional `bias_network: torch.nn.Sequential | None` to `TransgenerationalMemory`. `frozen=True`. `__post_init__` SHALL `copy.deepcopy` the caller-passed module and reassign via `object.__setattr__` (mirrors the existing `logit_bias.detach().clone()` clamp-clone pattern) so caller-side mutations cannot bleed across generations. `apply_to_logits(logits, sensory_input)` returns `logits + bias_network(sensory_input)` when set; falls back to `logits + self.logit_bias` when `bias_network is None` (M6 byte-equivalent). All existing call sites of `apply_to_logits` in `brain/arch/lstmppo.py` and `evolution/fitness.py` updated to pass `sensory_input` (legacy path ignores the arg).
- [x] 1.2 Extend `inherit_from(parents, decay_factor, decay_shape="geometric")`: branch on `decay_shape ∈ {"geometric", "linear", "sigmoid"}`. Geometric scales weight tensors by `decay_factor`; linear by `max(0, 1 − lineage_depth × (1 − decay_factor))`; sigmoid by `sigmoid(2 × (midpoint − lineage_depth))`. Default `"geometric"` preserves M6 cascade. All existing call sites (`evolution/loop.py`, `evolution/fitness.py`, M6 tests) MUST pass `decay_shape` explicitly or accept the default — adding a 3rd parameter with default value is backwards-compatible.
- [x] 1.3 Rewrite `extract_from_brain` to fit the bias-network MLP when `bias_network` is non-None: 10 deterministic action samples per probe (seeded on `extraction_seed`); 50 Adam epochs over `(sensory_input → logit_offset_from_uniform)`. Closed-form least-squares branch when `hidden_dim == 0`. Legacy `logit_bias` extraction preserved when `bias_network` is None.
- [x] 1.4 Extend `save()` / `load()` round-trip to persist `bias_network.state_dict()` AND `input_features` metadata. `load()` validates `input_features` are known `BrainParams` field names; raises `RuntimeError` with field-name diagnostic on mismatch.
- [x] 1.5 Add `TransgenerationalConfig.bias_network` Pydantic sub-block in `quantumnematode/utils/config_loader.py`: `hidden_dim: int ≥ 0`, `activation: Literal["tanh", "relu", "gelu"]`, `input_features: list[str]`. Validator accepts raw `BrainParams` field names AND derived `_sin` / `_cos` transforms (the runtime extracts the radian field and applies `math.sin`/`math.cos`; only radian-valued fields support the suffix). Validator rejects unknown stems with a helpful message listing supported fields. Defaults preserve M6 byte-equivalence (absent sub-block → legacy `logit_bias` path).
- [x] 1.6 Add `TransgenerationalConfig.decay_shape: Literal["geometric", "linear", "sigmoid"]` (default `"geometric"`). Validator rejects unknown values.
- [x] 1.7 Tests in `tests/.../agent/test_transgenerational_memory.py` (+18 cases): bias-network init / clamp on raw output / no-input-mutation / shape validation; **deepcopy in `__post_init__` (caller-side `net[0].weight.data.zero_()` does not affect the stored substrate)**; geometric / linear / sigmoid decay scaling correctness + monotonicity; `apply_to_logits` with sensory_input + broadcast + clamp; **`_sin` / `_cos` transform applied consistently in both `apply_to_logits` and `extract_from_brain` — runtime feature value matches `math.sin(params.predator_gradient_direction)` for `predator_gradient_direction_sin` and `math.cos(...)` for `_cos`**; bias-network round-trip preserves state_dict + input_features; `extract_from_brain` deterministic on seed; linear-projection closed-form path; M6 byte-equivalent legacy path with `bias_network=None`.
